什么是标签语义化？
标签语义化
合理的标签做合理的事情 例如：现在用dl来进行定义 dt放着我们要解释的词 dd里面进行解释 一系列列表的时候 第一想到的是ul 或者 ol 标题 h1-h6 无意义的时候 用div span 点击访问某一个地址的时候 a标签 段落 p标签

标签语义化的好处？

  1. 在没有css样式的时候，页面也可以正常的显示
  2. 利于维护（注释要写的完美，注释尽量不要用中文【特别在css文件 请尽量用英文】）
  3. 利于爬虫的抓取也就是SEO的优化（h1-h6是标题）重要标签包裹的地方就是重点，就是爬虫要抓取的地方（h1-h6，p，em，strong，title（即指标签 也指标签属性））
  4. 在特殊终端（纯文本格式，有声阅读），可以良好的显示
h1-->在页面上的权重很高 所以一个页面最重要的内容 放置在h1中 尽量减少h1的滥用 一般首页中 h1 我们会放置公司的logo 和 公司的名称 在无意义的时候 尽量使用div 和 span这两个标签 在控制文字样式的时候（例如加粗和斜体），我们尽量不使用标签来实现效果，而是使用css属性来实现效果 /*css*/ //js
